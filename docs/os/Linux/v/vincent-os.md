# Vincent OS

## Screenshot

![Screenshot](https://vincent-os.v38armageddon.net/img/index/mainDesktop.png)

## Description and history
Vincent OS is a operating system based on Arch Linux. Changing the default Linux kernel to linux-zen, using customized KDE Plasma Layout, adding PowerShell as default shell (keeping bash as well), using custom solution like Core LivePatch, a system hotpatch to fix critical bugs without rebooting.

Developed by v38armageddon.
